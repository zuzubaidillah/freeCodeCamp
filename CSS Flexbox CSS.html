Sejajarkan Elemen Menggunakan Properti justify-contentLulus Terkadang item fleksibel dalam wadah fleksibel tidak mengisi semua ruang di wadah. Adalah umum untuk ingin memberi tahu CSS cara menyelaraskan dan memberi ruang pada item fleksibel dengan cara tertentu. Untungnya, justify-contentproperti memiliki beberapa opsi untuk melakukan ini. Tapi pertama-tama, ada beberapa terminologi penting untuk dipahami sebelum meninjau opsi tersebut. Berikut adalah gambar yang berguna menunjukkan baris untuk mengilustrasikan konsep di bawah ini. Ingat bahwa pengaturan wadah fleksibel sebagai baris menempatkan item fleksibel berdampingan dari kiri ke kanan. Wadah fleksibel yang ditetapkan sebagai kolom menempatkan item fleksibel dalam tumpukan vertikal dari atas ke bawah. Untuk masing-masing, arah item flex disusun disebut sumbu utama . Untuk satu baris, ini adalah garis horizontal yang memotong setiap item. Dan untuk kolom, sumbu utama adalah garis vertikal melalui item. Ada beberapa opsi cara memberi spasi pada item fleksibel di sepanjang garis yang merupakan sumbu utama. Salah satu yang paling umum digunakan adalah justify-content: center;, yang menyelaraskan semua item fleksibel ke tengah di dalam wadah fleksibel. Pilihan lainnya termasuk: flex-start: menyelaraskan item ke awal wadah fleksibel. Untuk satu baris, ini mendorong item ke kiri wadah. Untuk kolom, ini mendorong item ke atas wadah. Ini adalah perataan default jika tidak ada justify-contentyang ditentukan. flex-end: menyelaraskan item ke ujung wadah fleksibel. Untuk satu baris, ini mendorong item ke kanan wadah. Untuk kolom, ini mendorong item ke bagian bawah wadah. space-between: menyelaraskan item ke tengah sumbu utama, dengan ruang ekstra ditempatkan di antara item. Item pertama dan terakhir didorong ke tepi wadah fleksibel. Sebagai contoh, berturut-turut barang pertama berhadapan dengan sisi kiri wadah, barang terakhir berhadapan dengan sisi kanan wadah, kemudian sisa ruang didistribusikan secara merata di antara barang-barang lainnya. space-around: mirip dengan space-betweentetapi item pertama dan terakhir tidak dikunci ke tepi wadah, ruang didistribusikan di sekitar semua item dengan setengah ruang di kedua ujung wadah fleksibel. space-evenly: Mendistribusikan ruang secara merata di antara item fleksibel dengan ruang penuh di kedua ujung wadah fleksibel Sebuah contoh membantu menunjukkan properti ini beraksi. Tambahkan properti CSS justify-contentke #box-containerelemen, dan berikan nilai center. Bonus Coba opsi lain untuk justify-contentproperti di editor kode untuk melihat perbedaannya. Tetapi perhatikan bahwa nilai centeradalah satu-satunya yang akan melewati tantangan ini.

Sejajarkan Elemen Menggunakan Properti align-items The align-itemsproperti mirip dengan justify-content. Ingatlah bahwa justify-contentproperti menyelaraskan item fleksibel di sepanjang sumbu utama. Untuk baris, sumbu utama adalah garis horizontal dan untuk kolom itu adalah garis vertikal. Wadah fleksibel juga memiliki sumbu silang yang merupakan kebalikan dari sumbu utama. Untuk baris, sumbu silang adalah vertikal dan untuk kolom, sumbu silang adalah horizontal. CSS menawarkan align-itemsproperti untuk menyelaraskan item fleksibel di sepanjang sumbu silang. Untuk satu baris, ini memberi tahu CSS cara mendorong item di seluruh baris ke atas atau ke bawah di dalam wadah. Dan untuk kolom, cara mendorong semua item ke kiri atau ke kanan di dalam wadah. Nilai berbeda yang tersedia untuk align-itemsmeliputi: flex-start: menyelaraskan item ke awal wadah fleksibel. Untuk baris, ini menyelaraskan item ke bagian atas wadah. Untuk kolom, ini menyelaraskan item ke kiri wadah. flex-end: menyelaraskan item ke ujung wadah fleksibel. Untuk baris, ini menyelaraskan item ke bagian bawah wadah. Untuk kolom, ini menyelaraskan item di sebelah kanan wadah. center: menyelaraskan item ke tengah. Untuk baris, ini menyelaraskan item secara vertikal (ruang yang sama di atas dan di bawah item). Untuk kolom, ini menyelaraskannya secara horizontal (ruang yang sama di kiri dan kanan item). stretch: meregangkan item untuk mengisi wadah fleksibel. Misalnya, item baris direntangkan untuk mengisi wadah fleksibel dari atas ke bawah. Ini adalah nilai default jika tidak ada align-itemsnilai yang ditentukan. baseline: menyelaraskan item ke garis dasar mereka. Baseline adalah konsep teks, anggap itu sebagai garis tempat huruf-huruf itu duduk. Sebuah contoh membantu menunjukkan properti ini beraksi. Tambahkan properti CSS align-itemske #box-containerelemen, dan berikan nilai center. Bonus Coba opsi lain untuk align-itemsproperti di editor kode untuk melihat perbedaannya. Tetapi perhatikan bahwa nilai centeradalah satu-satunya yang akan melewati tantangan ini.

Gunakan Properti flex-wrap untuk Membungkus Baris atau Kolom CSS flexbox memiliki fitur untuk membagi item fleksibel menjadi beberapa baris (atau kolom). Secara default, wadah fleksibel akan cocok dengan semua item fleksibel. Misalnya, satu baris semua akan berada di satu baris. Namun, menggunakan flex-wrapproperti memberitahu CSS untuk membungkus item. Ini berarti item tambahan dipindahkan ke baris atau kolom baru. Titik putus di mana pembungkus terjadi tergantung pada ukuran barang dan ukuran wadah. CSS juga memiliki opsi untuk arah pembungkus: nowrap: ini adalah pengaturan default, dan tidak membungkus item. wrap: membungkus item ke beberapa baris dari atas ke bawah jika berada dalam baris dan dari kiri ke kanan jika berada dalam kolom. wrap-reverse: membungkus item ke beberapa baris dari bawah ke atas jika berada dalam baris dan dari kanan ke kiri jika berada dalam kolom. Tata letak saat ini memiliki terlalu banyak kotak untuk satu baris. Tambahkan properti CSS flex-wrapke #box-containerelemen, dan berikan nilai wrap.

Gunakan Properti flex-shrink untuk Mengecilkan Item Sejauh ini, semua properti dalam tantangan berlaku untuk wadah fleksibel (induk dari item fleksibel). Namun, ada beberapa properti yang berguna untuk item fleksibel. Yang pertama adalah flex-shrinkproperti. Saat digunakan, ini memungkinkan item menyusut jika wadah fleksibel terlalu kecil. Item menyusut ketika lebar wadah induk lebih kecil dari lebar gabungan semua item fleksibel di dalamnya. The flex-shrinkproperti mengambil nomor sebagai nilai-nilai. Semakin tinggi angkanya, semakin menyusut dibandingkan dengan barang-barang lain di dalam wadah. Misalnya, jika satu item memiliki flex-shrinknilai 1dan yang lainnya memiliki flex-shrinknilai 3, item dengan nilai 3akan menyusut tiga kali lipat dari yang lain. Tambahkan properti CSS flex-shrinkke keduanya #box-1dan #box-2. Berikan #box-1nilai dari 1dan #box-2nilai 2.
